# Healthcare Digital Twin

**AI-Powered Healthcare Analytics Platform**

Healthcare Digital Twin is an intelligent healthcare analytics platform that leverages Machine Learning to assess heart disease risk, generate personalized health insights, recommend treatments, and visualize patient health trends through an interactive web dashboard.

The application provides secure authentication, role-based access for patients and doctors, health forecasting, AI-powered recommendations, PDF report generation, and patient history management.

# Project Overview

Healthcare Digital Twin simulates a patient's health profile using Machine Learning and generates actionable insights for healthcare monitoring.

The system enables:

* Heart Disease Prediction
* Personalized Health Score
* Risk Assessment
* AI-Based Recommendations
* Health Forecasting
* Doctor Dashboard
* Patient History Tracking
* PDF Report Generation

# Installation

Clone the repository

```bash
git clone https://github.com/umangvarshney-ops/Healthcare-Digital-Twin.git
```

Move into the project

```bash
cd Healthcare-Digital-Twin
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run dashboard/app.py
```
